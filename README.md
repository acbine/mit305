
# Team project 생산관리

## 프로젝트 소개
생산관리시스템의 일부를 요구사항에 맞게
학습용으로 구현


## 개발기간
2023.12 1주차~2023.02.1주차

### 개발환경

- Java17
- IDE:Intellij
- FrameWork:SpringBoot
- DB : MySQL
- ORM : JPA
